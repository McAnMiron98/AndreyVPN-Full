# AndreyVPN 1.0.3 — Compact Server List UX

## Changes

- Updated app version to 1.0.3+70.
- Made proxy/server list cards more compact.
- Reduced card height, margins, paddings and flag size.
- Changed server cards to a compact two-line layout.
- Lowest and balance entries now fit side-by-side on standard desktop widths.
- Added colored ping display:
  - green: 0–60 ms
  - yellow: 61–150 ms
  - red: 151+ ms
- Kept selected server marker: `✓ Подключен`.
- No VPN core logic changes.
