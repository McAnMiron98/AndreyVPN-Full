# AndreyVPN 1.0.2 — Server List UX

## Changes
- Improved proxy/server list cards for better readability.
- Added separate country line based on resolved server IP country.
- Server names now use two lines instead of being aggressively truncated.
- Leading duplicate flag emojis are removed from displayed server names.
- Added combined details line with connection status, protocol, security hint and ping.
- Selected server now shows `✓ Подключен`.
- Increased proxy grid card height and width for the new layout.
- Improved active proxy footer name wrapping and removed duplicate leading flags.
- No VPN core logic changes.
