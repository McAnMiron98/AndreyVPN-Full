# AndreyVPN 0.9.0

## Изменения

- Полная очистка проекта от оставшихся упоминаний старого бренда в текстовых файлах.
- Переименованы внутренние пути, пакеты, схемы, protobuf/options и служебные файлы с `andreyvpn` на `andreyvpn`.
- Обновлена версия проекта в `pubspec.yaml` до `0.9.0+41`.
- Очищены GitHub/CI/документация/README/история/конфиги от старого брендинга.

## Важно

Это большой ребрендинг внутренних имён. После распаковки рекомендуется выполнить чистую сборку:

```bash
flutter clean
flutter pub get
```
