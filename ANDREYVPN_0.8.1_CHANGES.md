# AndreyVPN 0.8.1

Добавлен полный импорт/экспорт приложения без изменения VPN-ядра и внутренней hiddify-core структуры.

## Что добавлено

- Новый пункт в меню настроек: **Полный бэкап**.
- **Экспорт полного бэкапа** в ZIP-файл.
- **Импорт полного бэкапа** из ZIP-файла.

## Что попадает в полный бэкап

- настройки приложения;
- база данных приложения;
- профили;
- локальные конфиги профилей;
- рабочие файлы portable-версии;
- Per-App Proxy данные и остальные локальные данные приложения.

После импорта рекомендуется полностью перезапустить AndreyVPN.
