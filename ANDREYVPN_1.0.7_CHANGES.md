# AndreyVPN 1.0.7 — Tray Menu Foreground Fix

## Что исправлено
- Исправлено поведение контекстного меню в трее Windows: теперь меню закрывается при клике вне области меню.
- Перед открытием native tray menu приложение делает своё Flutter-окно foreground owner через Win32 `SetForegroundWindow`.
- После возврата из `trayManager.popUpContextMenu()` отправляется `WM_NULL` через `PostMessageW`, чтобы Windows корректно завершала цикл popup-меню.
- Добавлена защита от двойного события правого клика по иконке трея.

## Технические детали
- Добавлен Windows-specific helper `lib/core/win32_tray_focus_fix.dart`.
- Изменён `SystemTrayNotifier.onTrayIconRightMouseDown()`.
- Логика подключения, системного прокси, автозапуска, импорта профилей и переключения серверов не менялась.

## Что проверить
1. Запустить AndreyVPN на Windows.
2. Нажать правой кнопкой мыши по иконке в трее.
3. Кликнуть вне меню — меню должно закрыться.
4. Проверить пункты меню: подключение/отключение, смена сервера, режим сервиса, выход.
5. Проверить левый клик по иконке трея: окно должно показываться/скрываться как раньше.
