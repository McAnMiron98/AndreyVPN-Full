# AndreyVPN 0.5.8

- Автообновление теперь запускается без окна PowerShell.
- Вместо консоли показывается небольшое окно прогресса обновления.
- После обновления окно прогресса закрывается автоматически.
- Обновлённый AndreyVPN запускается отдельно и не закрывается вместе с updater-ом.
- VPN-логика не изменялась.
